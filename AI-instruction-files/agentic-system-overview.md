# Agentic System Overview

The platform is composed of multiple AI agents.

Each agent owns a specific responsibility.

Agents collaborate to solve user requests.

---

Example Request:

"I need groceries for a family of four under $150."

---

Agent Workflow

User Request Agent: Understands the users' requests, and clarifies any ambiguities.

↓

Planner Agent : Decides what each agent needs to do, and in what order.

↓

Basket Agent : Identifies the products needed for the shopping list, and creates a comprehensive shopping list.

↓

Price Discovery Agent : Finds the prices of the products in the shopping list from various sources.

↓

Store Comparison Agent : Compares prices of products across different stores, and recommends the most cost-effective option.

↓

Savings Optimizer Agent : Recommends products based on user preferences and budget.

↓

Final Recommendation Agent : Presents the final recommendation to the user in a clear and concise manner.

---

Benefits:

- Better recommendations
- Dynamic reasoning
- Personalized suggestions
- Autonomous decision making
