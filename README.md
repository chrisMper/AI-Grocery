# Product Vision

## Product Name

GroceryAgent

## Vision

Build an AI-powered grocery shopping assistant that helps New Zealand shoppers spend less money on groceries.

Instead of manually comparing prices across supermarkets, users interact with an intelligent AI assistant.

The AI continuously monitors grocery prices, promotions, and shopping habits to recommend the most cost-effective purchasing decisions.

---

## Mission

Help users answer:

- Where should I shop this week?
- What is the cheapest grocery basket?
- Which products are currently discounted?
- What alternatives can save me money?
- When should I buy a product?

---

## Core Differentiator

This is not a grocery search engine.

This is an autonomous grocery intelligence platform powered by AI agents.

The platform actively helps users save money.
